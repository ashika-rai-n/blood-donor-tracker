# 🩸 Blood Donor Tracker

A web-based application to find and manage blood donors efficiently.

---

## 🚀 Features

* Search donors by blood group
* Location-based donor matching
* Donor availability tracking
* Blood request system

---

## 🌐 Live Demo

🔗 https://ashika-rai-n.github.io/blood-donor-tracker/

> ⚠️ Note:
> This project is deployed using GitHub Pages, which supports only static files (HTML, CSS, JavaScript).
> Backend features like login, registration, and database connectivity require running the server locally.

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Node.js

---

## 📂 Project Structure

* `index.html` → Home page
* `dashboard.html` → User dashboard
* `find.html` → Search donors
* `server.js` → Backend server

---

## ⚙️ Requirements

Make sure you have installed:

* Node.js
* npm (comes with Node.js)

---

## ▶️ How to Run the Project 

### 1️⃣ Download the Project

* Click **Code → Download ZIP**
* Extract the folder

---

### 2️⃣ Open in VS Code

* Open the folder in **VS Code**

---

### 3️⃣ Install Dependencies

Open terminal and run:

```bash
npm install
```

---

### 4️⃣ Start the Server

```bash
node server.js
```

---

### 5️⃣ Open in Browser

Go to:

```
http://localhost:3000
```

(or the port shown in terminal)

---

## ⚠️ Important Note

* `.env` file is not included for security reasons
* If required, create your own `.env` file
